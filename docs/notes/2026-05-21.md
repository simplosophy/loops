# 2026-05-21

## Shell tool optimization

- 参考 `openai-agents-python` shell tool 的结构化 request/output 设计，保留 loop0 内置本地执行模型。
- `shell.run` 新增 `commands` 序列、`timeout_ms`、`max_output_length`、`cwd`、`working_directory`、`env` 兼容输入。
- foreground 执行结果统一记录为 `command/stdout/stderr/status/outcome`，并写入 `ToolResult.metadata["outputs"]`；可读输出仍保持单条命令 stdout 兼容。
- 多命令执行采用 fail-fast 语义，遇到非零退出码、timeout 或取消立即停止。
- background session 新增 `list` 操作，`log` 改为按采集顺序返回 stdout/stderr 条目。
