## loop2 包结构

- `loops/loop2/_ids.py` — ULID + 前缀生成器（task_/ckpt_/rev_/art_/led_/aud_）。
- `loops/loop2/types.py` — `ProtocolError`（含错误码）、`Literal` 类型别名（TaskState 等）。
- `loops/loop2/objects.py` — 7 个一等对象 dataclass。不可变值对象 frozen（TaskSpec/OwnershipTransfer/CheckpointResolution/ReviewComment/ArtifactPayload/ArtifactRef/LedgerEntry），状态对象可变（Task/Checkpoint/Ownership/Review/Artifact/Ledger）。
- `loops/loop2/state_machine.py` — 显式合法转移表 `LEGAL_TRANSITIONS` + `check_transition` 校验，非法转移抛 `PRECONDITION_FAILED`。
- `loops/loop2/audit.py` — `AuditEvent`（frozen）+ `AuditLog`（append-only，单调 seq，支持 query/replay）。
- `loops/loop2/store.py` — `HACPStore` 内存存储，按 id 索引各对象，artifacts 按 (id, version) 二元组建索引。
- `loops/loop2/contracts.py` — `AAPBridge` Protocol（HACP→AAP 层间接口）+ `InMemoryAAPBridge` stub（记录调用不执行，验证 TaskID 贯穿）。
- `loops/loop2/operations.py` — `HACPOperations` facade，21 个操作统一结构：前置条件→状态转移→audit→层间契约。
- `loops/loop2/__init__.py` — 公开 API re-export。

## 关键实现决策

- **风格对齐 loop0**：stdlib dataclasses + `typing.Literal`（非 enum）+ `from __future__ import annotations`，无 pydantic。
- **分层纪律**：loop2 刻意不 import loop1/loop0，证明协议层可独立存在（transport-agnostic），有 `test_loop2_does_not_import_lower_layers` 守护。
- **测试风格**：遵循 `tests/test_loops_core.py` 约定——同步 `def test_*` + `asyncio.run()` 驱动 async 操作，不引入 pytest-asyncio。
- **状态机修正**：`review_submit` 先 `review_ready→under_review` 再按 verdict 转，对应 spec §3.3 合法路径。
- **CLI 本阶段不做**：先验证协议语义，CLI 留后续。
