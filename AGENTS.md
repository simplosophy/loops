# AGENTS.md

Loops Agent 是一个用于用于构造复杂Agent的框架。

## 强制
- **KEEP UPDATE**: 如果较大代码变更，及时使用git提交代码，comment中包含变化的详细描述
- 遵守架构设计文档`docs/architecture/OVERVIEW.md`，如果修改对架构有改造，需要进行讨论并确认是否符合设计原则
- **critical** 较大的修改重构按照日期记录在 `docs/notes/yyyy-MM-dd.md`

## 设计原则

- **极简**: 保持核心功能简单，避免复杂配置
- **正交性**: 保持核心功能正交，unix设计思想，每个工具做一件事情，并且做好
- **一致性**: 代码风格和命名规范一致
- **分层架构**: 层与层之间通过接口通信
- **Fail Fast**: 快速失败，避免长时间阻塞
- **约定>配置**: 优先使用约定而非复杂配置
